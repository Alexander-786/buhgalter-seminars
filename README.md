# 📚 Веб-сторінка для семінарів бухгалтерів

## Опис

Професійна веб-сторінка для управління та відображення онлайн-семінарів з бухгалтерії. Розроблена для Вікторії Величко.

## 🎯 Основні можливості

### Для відвідувачів:
- 📋 Красивий список всіх семінарів
- 📅 Детальна інформація про кожен семінар (дата, час, тема, лектор)
- 💰 Система ранніх цін зі зворотним відліком
- 📱 Адаптивна верстка (мобільні пристрої, планшети, комп'ютери)
- 📞 Контактна інформація

### Для адміністратора:
- 🔧 Приватна панель управління
- ➕ Додавання нових семінарів
- ✏️ Редагування існуючих семінарів
- 🗑️ Видалення семінарів
- 💾 Автоматичне збереження даних у браузері

## 🚀 Швидкий старт

### Локально:

1. Завантажте репозиторій
2. Відкрийте файл `index.html` у браузері
3. Натисніть кнопку "🔧 Адміністратор" для управління семінарами

### На GitHub Pages:

1. Перейдіть в **Settings** репозиторію
2. Знайдіть розділ **Pages**
3. Виберіть **Source**: Main branch
4. Ваша сторінка буде доступна за URL

## 📁 Структура файлів

```
buhgalter-seminars/
├── index.html       # Основна HTML-сторінка
├── styles.css       # Стилізація (дизайн)
├── script.js        # JavaScript логіка
└── README.md        # Цей файл
```

## 🛠️ Як користуватися панеллю адміністратора

1. **Натисніть "🔧 Адміністратор"** у верхньому правому куті
2. **Заповніть форму** для додавання нового семінару:
   - Назва семінару *(обов'язково)*
   - Дата у форматі дд.мм.рррр *(обов'язково)*
   - Час початку *(обов'язково)*
   - Місце проведення
   - Опис
   - Вартість *(обов'язково)*
   - Рання ціна (опціонально)
   - Дедлайн ранної ціни (опціонально)
   - Лектор
3. **Натисніть "➕ Додати семінар"**

### Редагування та видалення:

1. Натисніть на картку семінару
2. У модальному вікні натисніть:
   - **"✏️ Редагувати"** - для редагування
   - **"🗑️ Видалити"** - для видалення

## 💾 Де зберігаються дані

Усі дані зберігаються **у браузері** (localStorage):
- Нічого не завантажується на сервер
- Дані збережуться, навіть якщо закрити браузер
- Дані видаляться, якщо очистити кеш браузера
